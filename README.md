# Segurança Cibernética no Setor Financeiro

Projeto acadêmico desenvolvido utilizando NotebookLM para construção de um caderno temático com apoio de IA Generativa.

## Objetivo
Analisar riscos cibernéticos no setor financeiro brasileiro, utilizando fontes oficiais e técnicas de engenharia de prompt.

## Fontes
- Banco Central do Brasil
- Febraban
- CVM
- ANBIMA

## Estrutura do Projeto
- Objetivos de estudo
- Perguntas estratégicas
- Testes de prompts
- Miniguia estruturado
- Glossário de termos técnicos

## Competências Desenvolvidas
- Curadoria de fontes
- Pensamento crítico
- Engenharia de prompt
- Organização de documentação técnica
